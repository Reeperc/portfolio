Network Architecture Project

Project completed at the end of the 4th year of engineering school

Set up a secure local network for a gaming championship using the SCRUM methodology.

Project Description

  • Set up the network and system infrastructure
  • Implement network security (distribution and segmentation of the class C IP address 195.221.X.0 into 3 subnets)
  • Develop scripts for remote competition launches

Proposed Solution

Network Part:
  • A router to interconnect the city with the rest of the world
  • An IP addressing plan
  • A backup of the configuration on a server
  • LAN traffic security
System Part:
  • A web server for displaying scores and launching games
  • A mail server and a webmail for appointment messages
  • Two DNS servers (Linux and Windows) for resolving DMZ and Trusted names
  • An Active Directory directory server to authenticate players
Match Management:
  • A secure web interface for displaying results (protected by directory login/password)
  • The ability to launch a server to host a competition (Administrator)
  • To prevent cheating, the player must have the game and only the game on their screen

Deliverables

  • Network plan
  • Installation procedures for each equipment
  • Database schema
  • Web application (PHP)
  • Test scenario

Educational Objectives

  • Identify and describe the different roles in a SCRUM project
  • Apply and understand the agile SCRUM methodology in a project
  • Participate in a project in agile mode
  • Deploy network services
  • Develop a web application
  • Know the basic IOS commands
  • Increase shell knowledge

Prerequisites

  • TCP/IP Networks
  • Cisco IOS
  • Linux Shell
  • Web
  • PHP
  • VirtualBox
  • Windows Server
  • Apache
  • SMTP
